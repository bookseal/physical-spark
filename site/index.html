The real setup

The dream is a toy. The tool is real.

No RC car, really — the missions run on the SO-101, the low-cost, open-source 3D-printed robot arm from the Hugging Face LeRobot ecosystem. Learn it in simulation on your own laptop first; buy the arm only when you're ready.

6-DOF · open-source (Apache-2.0) · a few hundred dollars · Mac-friendly sim path
