# 밤샘 빌드 로그 — 코스 + 게임 (overnight session)

> 창업자가 자는 동안 Claude가 자율로 빌드한 기록. 계획 → 시도 → 됨/안됨 → 결정을 남긴다.
> 아침에 읽는 용도. 커밋은 마일스톤마다, 안 되면 되돌릴 수 있게.

## 목표 (창업자 지시)
1. **코스들을 직접 만들기** — 동적으로 화려하게. 마지막 프로젝트 **"무궁화 꽃이 피었습니다"**(Red Light, Green Light)까지.
2. **게임을 적당히 즐길 수 있게** — 처음에 시선 끌고 빠지게. Re-Volt식 **미끄러지는 타이어 자국**.
3. 중간중간 commit, 안 되면 롤백. 시도 흔적을 .md로.

## 계획 (내 로드맵)
- **A. 서버 정적 서빙** — 뷰어가 `site/` 하위(courses 등)를 서빙하게 (GitHub Pages와 동일 동작).
- **B. 코스 엔진** — `site/courses/index.html`: JS 데이터 기반 동적 코스 앱. 해시 라우팅(#/m1), 진도 저장(localStorage), 화려한 웜 테마.
  - 3개 미션: (1) 토닥토닥 (2) pick&place (3) 무궁화꽃. 각: 히어로 → 배울 것 → 레벨/시간/장비 태그 → 레슨 스텝(개념+코드+체크포인트) → 진도바 → 다음.
  - 내용은 lab/tutorials·foundations·course-analysis에서 grounded. **v0 초안**임을 정직하게.
- **C. 랜딩 연결** — 미션 카드 → 코스 링크.
- **D. 게임 개선** — 타이어 스키드 자국(드리프트 트레일), 점수 juice, 시선 끌기. "적당히".
- **E. 로그·커밋·테스트** — 마일스톤마다 스샷+커밋, 이 로그 갱신.

## ☀️ 아침 요약 (일어나서 읽어)
**다 됐고 커밋됨. 롤백 안 해도 됨.** 실물 확인은 **Playbook의 🚀 Landing → ▶ Open the full curriculum** 또는
`site/index.html`·`site/courses/` 직접 열기.
- **코스**: `site/courses/` — 3미션 동적 앱(토닥토닥→pick&place→**무궁화꽃/Red Light**). 진도 체크(브라우저 저장),
  레슨 아코디언에 개념+코드+체크포인트. **내용은 v0 초안** — 네가 실제로 해보면서 채워야 할 부분엔 ⚠️note로 정직하게 표시함.
- **게임**: 랜딩 "Take a lap". ←→/드래그로 조작, 블록 피하고 별 먹기, 드리프트 스키드 자국. **적당히 중독성**.
- **직접 해볼 것**: ① 게임 한 판(스샷으론 조작감 검증 못 함) ② 코스 레슨 체크박스 눌러 진도바·토스트 동작 확인.
- **다음 후보**: 코스 내용 심화(네 실전 경험 반영), SO-101 생성이미지 코스에 삽입, 게임에 사운드/모바일 버튼 다듬기.

### 안 한 것 / 주의
- 코스 내용은 **실제로 안 해본 v0** — 특히 무궁화꽃은 개념 스캐폴드다. 네가 시뮬 돌려보고 고쳐야 진짜가 됨.
- 게임은 "적당히"에 맞춰 사운드·파티클은 최소. 원하면 더 juicy하게.

## 밤샘 2차 — 게임 재미 요소 (Re-Volt 후기 기반)
- 리서치: Re-Volt의 재미 = **Mario-Kart식 픽업(배터리 부스트⚡·불꽃·전기·오일)** + RC 무게감 물리 + 향수.
  (출처: revolt.fandom Pickups, GameSpot/Metacritic 리뷰 — "Best Racing Game 1999")
- [게임 v2 ✅] 우리 러너에 이식:
  - **⚡ 배터리 부스트 픽업** — 먹으면 1.7배 속도·부스트미터·속도선·글로우·whoosh 사운드 (Re-Volt 시그니처).
  - **★ 콤보** — 픽업 연속 시 x배수, 팝업, 콤보 사운드.
  - **니어미스("CLOSE!")** — 블록 아슬아슬 통과 시 보너스+사운드 (긴장/보상).
  - **3-2-1 GO 카운트다운**, **마일스톤(500m마다 팡파레+플래시)**, **파티클 버스트**(픽업/크래시), **차체 부스트 글로우**.
  - **WebAudio 합성 사운드** — 엔진음(속도따라 피치)·픽업·부스트·크래시·카운트다운. **파일 0, 저작권 0.** 음소거 버튼(🔊/🔇).
  - 첫 입력 전엔 무음(autoplay 정책 준수), 첫 조작에 오디오 init.
  - 상태머신: ready(attract)→count(3-2-1)→run→dead. 재시작은 카운트다운 스킵.
- **주의**: 사운드·부스트·콤보·카운트다운은 **정적 스샷으로 검증 불가** → 아침에 직접 플레이해서 확인 필요. JS는 node --check 통과, 렌더 정상.
- 다음(선택): 오일 슬릭 해저드(RC 미끄러짐), 신기록 축하, 난이도 곡선 튜닝.
